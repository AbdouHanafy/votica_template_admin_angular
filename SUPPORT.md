# Support Policy

Product: **Votica Template Admin Dashboard**  
Author: **Abdou Hanafi**

## Included Support

- Installation and local setup help
- Build and dependency troubleshooting
- Bug fixes for issues reproducible in the distributed template
- Guidance for theme variables and basic customization

## Not Included

- New custom features outside the original template scope
- Backend/API implementation
- Third-party plugin paid support
- Full custom project development

## Response Time

- Target first response: **within 24 to 48 hours** (business days)

## Support Channel

- ThemeForest comments/support tab (recommended)
- Email support (if provided in marketplace profile)

## Information Required for Fast Support

- Angular version and Node/NPM versions
- Exact error message and stack trace
- Steps to reproduce
- Screenshots or short screen recording
